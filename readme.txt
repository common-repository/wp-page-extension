This plugin will allow you to add extensions to the end of your pages. Choices are: .html, .htm, .php, .xml, .shtml and .asp.

== Description ==

Converting an old HTML, PHP, or ASP site to WordPress? WP Page Extension will let you create pages inside of wordpress and give them any extension you want. It will help your site never lose any old backlinks and seo. You can choose any page to have any extension. Choices are: .html, .htm, .php, .xml, .shtml and .asp. If you dont pick an extension then it will just default to no extension.

== Installation ==

1. Upload `wp-page-extension` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Edit whatever page you want an extension on
